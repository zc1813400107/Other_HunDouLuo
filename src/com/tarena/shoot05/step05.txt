从图-15可以看出，要实现英雄机跟随着移动而移动，那么英雄机的坐标算法如下：
hero.x=鼠标的x坐标-width/2；
hero.y=鼠标的y坐标-height/2
moveTo方法的代码如下所示：
	/**
	* 当前物体移动了一下，相对距离， x,y鼠标位置
	*/
	public void moveTo(int x, int y) {
		this.x = x - width / 2;
		this.y = y - height / 2;
	}
